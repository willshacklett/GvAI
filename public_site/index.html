GvAI Public Signal Stream

This is a public machine-readable recoverability and trajectory signal stream.

Endpoints

  • latest.json — current state
  • history.json — full history
  • schema.json — field definitions
  • manifest.json — metadata
  • llms.txt — model instructions

Interpretation

  • Prefer trajectory over snapshot.
  • Use history.json for trend and persistence analysis.
  • Use latest.json for current-state integrations.
  • lead_time represents early warning distance before collapse.
